
This folder allows the website to override what is sent from the application.

This gives the site greater flexibility in what the user should see based on the situation.

- 404 = Unknown pages.
- 500 = Internal/Development Error, Unable to handle the eror.
- disabled = Pages that are disabled from view on the dashboard
- database = Database issues stop the application from loading, usually relate to the database.
- maintenance = The website is offline, display friendly message to represent that.
